# バージョン情報

## 現在のバージョン: v1.0.0

## バージョン管理

このリポジトリは[セマンティックバージョニング](https://semver.org/lang/ja/)に従ってバージョン管理されています。

### バージョン確認方法

```bash
# 現在のバージョンを確認
cd development-rules
git describe --tags --always

# 特定のバージョンに切り替え
git checkout v1.0.0
```

### バージョン更新の確認

```bash
# リモートの最新タグを確認
git fetch --tags
git tag -l

# 最新バージョンに更新
git submodule update --init --recursive --remote
```

### バージョン固定

安定した開発環境を維持するため、特定のバージョンに固定することを推奨します。

```bash
# 特定のバージョンに固定
cd development-rules
git checkout v1.0.0

# プロジェクトのルートに戻ってコミット
cd ..
git add development-rules
git commit -m "fix: development-rulesをv1.0.0に固定"
```

## セマンティックバージョニング

### バージョン番号の形式: MAJOR.MINOR.PATCH

- **MAJOR**: 破壊的変更（APIの変更など）
- **MINOR**: 新機能の追加（後方互換性あり）
- **PATCH**: バグ修正（後方互換性あり）

### 例

- `v1.0.0`: 初回リリース
- `v1.1.0`: 新機能追加（例：新しいコマンド）
- `v1.1.1`: バグ修正
- `v2.0.0`: 破壊的変更（例：コマンド仕様の変更）

## リリースプロセス

1. **機能開発**: 新しい機能や修正を開発
2. **テスト**: すべての機能が正常に動作することを確認
3. **ドキュメント更新**: README.md、CHANGELOG.mdを更新
4. **タグ作成**: 適切なバージョン番号でタグを作成
5. **リリース**: タグをプッシュしてリリース完了

## サポートポリシー

- **最新バージョン**: 完全サポート
- **前のメジャーバージョン**: セキュリティ修正のみ
- **それ以前**: サポート終了
